# showbox

> [!WARNING]
> This project is not fully implemented.

showbox is a system for show ticketing.

## Installation

```bash
## Clone repository
git clone https://github.com/krapie/showbox.git

## Change directory
cd showbox

## Build
make build
```

## Usage

```bash
## Run showbox API server
./bin/showbox server
```
